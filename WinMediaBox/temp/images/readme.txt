NOTE:

Do not remove folders and files in images folder root

radio, twitch, yt - those folders using for covers, you can read about them in readme inside "data" folder

bg.jpg - this picture using as a background of app
default.png - this picture using in cases where was not assigned any other
radio-ani.gif - this is animated background for radio station screen
refresh.png - picture for update button

You can customize and replace those images (bg.jpg, default.png, radio-ani.gif, refresh.png), names of your custom images must be same as originals, dont forget to save original files to have a possibility to restore, valid extensions: must be same as in original files
